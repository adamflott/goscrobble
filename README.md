# goscrobble

Scrobble to (last.fm)[http://last.fm] from the command line. Written
in Go and requires no additional dependencies.

## Usage

    ./goscrobble --help
    Usage of ./golastfm:
      -album="": Track album
      -api_key="": API Key
      -api_secret="": API Secret
      -artist="": Track artist
      -session="": Session
      -title="": Track title
      -token="": Token
